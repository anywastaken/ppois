Лабораторная работа №2: Приложение для учета товаров
Назначение

Приложение с графическим интерфейсом для автоматизации учета товаров на складе. Позволяет управлять базой данных (добавление, поиск, удаление), просматривать записи в табличном виде с поддержкой постраничной навигации и осуществлять импорт/экспорт данных через XML-файлы.
Технологии

    Python 3.10+

    PyQt6 — библиотека для создания графического интерфейса (GUI).

    XML-парсеры:

        xml.sax (SAX) — событийный парсер для быстрого чтения данных.

        xml.dom.minidom (DOM) — построение объектной модели для записи данных.

    Архитектурный паттерн: MVC (Model-View-Controller).

XML-парсеры

    SAX-парсер: Реализован в классе ProductHandler (модуль model/Database.py). Используется для последовательного чтения XML-файла и загрузки товаров в список объектов Product.

    DOM-парсер: Используется в методе save_to_xml. Формирует древовидную структуру документа в оперативной памяти и сохраняет её в файл с форматированием (отступами).

Структура проекта и классы
Файл	Класс / Функция	Назначение
main.py	main()	Точка входа: инициализация слоев MVC и запуск цикла событий QApplication.
controller/Controller.py	Controller	Логика приложения: обработка сигналов от кнопок, управление пагинацией и вызов методов модели.
model/Product.py	Product	Модель данных: описание атрибутов товара (Название, Производитель, УНП, Кол-во, Адрес).
model/Database.py	Database	Хранилище данных: методы CRUD (создание, чтение, поиск, удаление) и работа с XML.
view/MainWindow.py	MainWindow	Главное окно приложения: меню, таблица и кнопки навигации.
view/AddProductDialog.py	AddProductDialog	Диалоговое окно для ввода данных о новом товаре.
view/SearchProductDialog.py	SearchProductDialog	Интерфейс расширенного поиска с собственной пагинацией результатов.
view/DeleteProductDialog.py	DeleteProductDialog	Окно для удаления группы записей по заданным фильтрам.
Данные (Вариант №6)

    Атрибуты товара:

        Название товара (str)

        Производитель (str)

        УНП (int, 9 знаков)

        Количество на складе (str/int)

        Адрес склада (str)

    Файлы:

        data.xml — основная база данных (50+ записей).

        data_2.xml — альтернативный набор данных для тестирования импорта.

Запуск приложения

Для запуска проекта убедитесь, что установлена библиотека PyQt6:
Bash

# Установка зависимостей
pip install PyQt6

# Запуск приложения из корневой директории
python main.py

Основные возможности

    Постраничный вывод: Отображение строго по 10 записей на страницу.

    Навигация: Кнопки перехода «В начало» (<<), «Назад» (<), «Вперед» (>) и «В конец» (>>).

    Гибкий поиск: Фильтрация записей по любому сочетанию атрибутов.

    Безопасный ввод: Использование QSpinBox гарантирует корректность числовых данных (УНП).
