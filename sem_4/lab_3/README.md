# Asteroids Python Edition

Небольшая аркадная игра на `pygame`, вдохновленная классической `Asteroids`.
Игрок управляет кораблем, уничтожает астероиды и НЛО, собирает бонусы, проходит уровни и пытается попасть в таблицу рекордов.

## Возможности

- главное меню с разделами `Start Game`, `High Scores`, `Help`, `Exit`
- управление кораблем с инерцией, поворотом и стрельбой
- генерация астероидов со случайной формой и разбиением на более мелкие
- появление НЛО с собственной логикой движения и стрельбы
- бонус `shield` с временной защитой
- бонус `rapid_fire` с ускоренной стрельбой
- бонус `double_score` с удвоением очков
- уровни с постепенным ростом сложности
- таблица рекордов с сохранением в `JSON`
- звуковые эффекты и фоновая музыка
- простые визуальные эффекты взрывов через частицы

## Требования

- Python `3.10+`
- библиотека `pygame`

## Установка и запуск

1. Установить зависимости:

```bash
pip install pygame
```

2. Запустить игру из корня проекта:

```bash
python main.py
```

## Управление

- `W` или `Up` - ускорение корабля
- `A` или `Left` - поворот влево
- `D` или `Right` - поворот вправо
- `Space` - выстрел
- `Esc` - возврат в меню

## Игровая логика

### Основные состояния

Приложение работает как конечный автомат со следующими экранами:

- `MENU` - главное меню
- `GAME` - игровой процесс
- `HIGHSCORES` - таблица рекордов
- `HELP` - экран с подсказками
- `INPUT_NAME` - ввод имени после нового рекорда

### Очки

Очки начисляются за уничтожение объектов:

- большой астероид - `20`
- средний астероид - `50`
- маленький астероид - `100`
- большое НЛО - `100`
- маленькое НЛО - `200`

Если активен бонус `double_score`, все получаемые очки удваиваются.

### Уровни

- стартовый уровень после начала игры: `1`
- переход на следующий уровень происходит каждые `1000` очков
- с ростом уровня ускоряется появление врагов за счет коэффициента `progression_coefficient`
- НЛО начинают появляться со `2` уровня

### Жизни

- начальное число жизней: `3`
- после потери жизни корабль возрождается в центре экрана
- после возрождения кратковременно активируется щит

### Бонусы

- бонус появляется при наборе определенного количества очков
- базовая точка появления первого бонуса: `500`
- длительность действия бонуса: `7000` мс
- время жизни бонуса на карте: `5000` мс

## Конфигурация

Основные параметры игры хранятся в [configs/config.json](configs/config.json).

Через конфиг можно изменить:

- размеры окна и `FPS`
- физику корабля и пуль
- количество и размеры астероидов
- параметры уровней
- шанс появления маленького НЛО
- длительность и типы бонусов
- количество жизней
- громкость музыки и звуковых эффектов

Примеры текущих настроек:

- размер окна: `1200x900`
- `FPS`: `60`
- стартовое число астероидов: `5`
- задержка выстрела: `250` мс
- стартовое число жизней: `3`

## Таблица рекордов

Рекорды сохраняются в [configs/highscores.json](/home/anywastaken/projects/university/sem_4/ppois/lab_3/configs/highscores.json).

Особенности:

- хранится до `10` лучших результатов
- записи сортируются по убыванию очков
- после нового рекорда игроку предлагается ввести имя длиной до `10` символов

## Как работает программа

1. В `AsteroidsGame.__init__()` инициализируются `pygame`, окно, звук, конфиг и игровые объекты.
2. В методе `run()` запускается основной цикл.
3. `handle_events()` обрабатывает клавиатуру, мышь и переходы между экранами.
4. `process_input()` отвечает за удерживаемые клавиши во время игры.
5. `update()` обновляет объекты, спавнит врагов и бонусы, проверяет столкновения и счет.
6. `draw()` рисует текущий экран и интерфейс.
